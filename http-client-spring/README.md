## Building and Running

1. Navigate to the project directory:

```
cd <projects_directory>/framework-examples/http-client-spring
```

2. Build the project:

```
mvn clean install
```

3. Run the application:

```
java -jar ./target/http-client-spring-1.0.0-SNAPSHOT.jar
```
